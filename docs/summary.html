CLOSURE

一小时之后留下什么

把七讲收束成四个问题：怎么动手、怎么记住、怎么分工、怎么控风险。

01

看懂结构

Agent = 模型 + 工具 + 循环 + 上下文 + 边界。看代码时，先找它怎么把目标拆成可执行动作。

02

判断能力

Rule 管约束，Skill 管方法，MCP 接工具；Memory 和压缩管上下文，SubAgent 和 Teams 管分工。

03

继续实践

想做项目，先跑完整代码；想补原理，再回到大模型系列。
